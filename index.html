How to use

  1. Set clinical context. Choose age, stage, optional APOE.
  2. Configure Test A. Pick a modality & category (Pos/Indet/Neg), edit LRs if needed.
  3. Optionally enable Test B. Posterior from A becomes the prior for B.
  4. Autopsy panel. We re-express your A/B LRs vs PET onto the autopsy scale and show an autopsy-anchored posterior.
  5. Prognosis. By default, uses the autopsy-anchored posterior.

    About this tool

    What this tool does

    This Bayesian Amyloid Helper computes posterior probability of amyloid positivity using Bayes theorem on odds ratios. It supports sequential testing with up to two biomarker tests and provides both PET-referenced and autopsy-anchored probability estimates for clinical decision-making.

    Key Features:

    • Age and APOE genotype-adjusted priors
    • Sequential Bayesian updating with likelihood ratios
    • Multiple biomarker modalities (PET, CSF, plasma)
    • Autopsy harmonization for cross-reference validity
    • Prognostic risk modeling with hazard functions

    Mathematical Foundation

    Bayes on Odds:
    prior_odds = p/(1−p)
    post_odds = prior_odds × LR
    post_p = post_odds/(1+post_odds)

    Sequential testing multiplies likelihood ratios, assuming conditional independence given disease status.

    Best Practices

    Combining tests: Bayes multiplication assumes tests are conditionally independent given disease. Many biomarker pairs are correlated (e.g., plasma tau + CSF tau), which can overstate evidence. Prefer mixing different pathways (e.g., plasma tau + CSF amyloid) or consider down-weighting correlated tests (LRα with α≈0.6–0.8).

    Autopsy‑Anchored Posterior: Methods & Sources

    Method note (click to toggle)

    We compute post‑test probabilities via Bayes in odds form using clinical priors (age, stage, ± APOE) and biomarker‑specific likelihood ratios (LRs) typically reported against in‑life references (Aβ‑PET or CSF). We then recalibrate from PET/CSF‑referenced log‑odds to an autopsy‑anchored posterior probability using published PET⇄autopsy performance (sensitivity/specificity) as a bridging step.
    Assumption: conditional independence of the Test and PET given autopsy status. When biomarkers are correlated, reduce the LR magnitude accordingly or use dependence‑aware fusion.
